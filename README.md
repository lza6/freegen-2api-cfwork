# freegen-2api-cfwork
Cloudflare Worker 边缘计算单文件部署、完美兼容 OpenAI 标准 API (支持 SSE 流式输出与 Vision 多模态图生图)、高级请求伪装与反检测 (内置高权重浏览器 Headers 绕过基础风控)、WebSocket 异步轮询与 SHA-256 动态签名鉴权、超大 Base64 图像 TCP 分块截断修复 (流式 Buffer 智能拼接)、内置 SaaS 级高颜值 WebUI (暗黑玻璃拟物化、支持拖拽与全局快捷键粘贴上传)、异步非阻塞处理 (基于 ctx.waitUntil 的性能优化)、全自动跨域 CORS 预检处理
